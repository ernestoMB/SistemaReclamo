# SistemaReclamo
Prueba técnica de un istema de ingreso y clasificación de reclamos


prueba técnica de conocimientos aplicados en C# ASP.Net en MVC
------------------------------------------------------------------------------------------------
Estructura:
- Base de datos relacional SQLServer
- Logica de programación
- Modelos
- Vistas
- Controladores
------------------------------------------------------------------------------------------------
Funcionalidad:
Cliente:
Debido a su incorformidad con el producto adquirido, el cliente puyede levantar un reclamo 
donde puede detallar proveedores, direcciones, montos, detalles del producto, etc.
Este sitio es administrado por una organización que se dedica a hacer valer derechos de los consumidores.
Empleado:
El empleado puede acceder a la plartaforma y ver el listado de reclamos activos que hacer en el sistema,
Ahí puede clasificarlos como:
Válido como reclamo
Aviso de infraccion
Asesoria
Además de verificar información completa del reclamo.

* C#
* Asp.Net
* Entity framework
* SqlServer
* HTML, CSS y JS
